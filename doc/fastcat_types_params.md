**schmitt_trigger**

| Parameter | Description |
| --------- | ----------- |
| output | either 1 or 0 depending on value of input signal |

**filter**

| Parameter | Description |

| Parameter | Description |
| --------- | ----------- |

| fz | force in z direction |

| ty | torque about y axis |
| tz | torque about z axis |
